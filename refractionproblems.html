When light strikes a boundary plane normal to the surface interface, the angle of incidence relative to the normal line reads exactly 0°. Despite hitting a medium change with different optical densities (n1 ≠ n2), the ray proceeds with zero angular deviation.

Snell's Law: n1 × sin(θ1) = n2 × sin(θ2) ⇒ sin(θ2) = 0 ⇒ θ2 = 0°

Advanced Optical Core: Even though the direction remains constant, refraction has structurally occurred. Because the velocity changes (v2 = c / n2), all internal wavefront segments decelerate or accelerate simultaneously along the boundary line. No phase lead or lag is generated across the wavefront profile, keeping the path completely straight.

Case 2: Homogeneous Refractive Index Matrix (n1 = n2) തുല്യ അപവർത്തന സൂചികയുള്ള മാധ്യമങ്ങൾ

When the optical density profiles of both optical media are perfectly matched (n1 = n2), the phase boundary functionally disappears for the passing wave packet. You can adjust the incidence slider to any angle, and the ray will never bend.

Since n1 = n2 ⇒ sin(θ1) / sin(θ2) = n2 / n1 = 1 ⇒ θ1 = θ2

Advanced Optical Core: This simulation illustrates optical cloaking or complete index matching. Because the structural phase speed does not shift when crossing the interface (v1 = v2), no phase distortion occurs across the wavefront, making the boundary completely transparent to the light path.

Case 3: Wave Propagation Velocity Invariance (v1 = v2) തുല്യ വേഗതയുള്ള സാഹചര്യങ്ങൾ (സ്ഥിര തരംഗ പ്രവേഗം)

By definition, an electromagnetic wave's phase velocity within an isotropic medium depends strictly on its absolute refractive index (v = c / n). Therefore, stating that light maintains an identical speed across an interface (v1 = v2) mathematically requires that their refractive indices match.

v1 = v2 ⇒ c / n1 = c / n2 ⇒ n1 = n2

Advanced Optical Core: This highlights a key concept in physics: light bends because its velocity changes across different media. If the propagation velocity remains completely unchanged across the boundary (v1 = v2), no wavefront bending forces can exist. Thus, angular deviation cannot occur.
